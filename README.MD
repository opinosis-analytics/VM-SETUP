This repository contains the shell script that installs the necessary library for the following projects

1. [Opinosis MultiLLM](https://github.com/opinosis-analytics/Opinosis-MultiLLM)

2. [Opinosis Multiparser](https://github.com/opinosis-analytics/Opinosis-MultiParser)

* To set up VM for [Opinosis MultiLLM](https://github.com/opinosis-analytics/Opinosis-MultiLLM) , see the steps in this readme [MultiLLM_VM_Setup.MD](./MultiLLM_VM_Setup.MD)

To set up VM for [Opinosis Multiparser](https://github.com/opinosis-analytics/Opinosis-MultiLLM) see the steps in this readme [MultiParser_VM_Setup.MD](./MultiParser_VM_Setup.MD)
